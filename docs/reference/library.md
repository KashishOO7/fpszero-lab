# The Library

> I read thousands of tabs so you don't have to. These are the ones worth your time.

Everything here is filtered for signal. If it's on this list, it's because it is genuinely one of the best resources in its category — not because it's popular or because it showed up in a "Top 10" list.

---

## Foundational Books

These are the books that give you the underlying theory — not just how to use a tool, but why the system works the way it does.

| Book | Domain | Why It's Here |
| :--- | :--- | :--- |
| [The Web Application Hacker's Handbook](https://portswigger.net/web-security/web-application-hackers-handbook) | Web | Still the most comprehensive treatment of web app attacks |
| [Hacking: The Art of Exploitation (Erickson)](https://nostarch.com/hacking2.htm) | Binary/Systems | Teaches C, assembly, buffer overflows from first principles |
| [Linux Journey (Interactive)](https://labex.io/linuxjourney) | Linux | Clean, interactive Linux fundamentals |
| [Practical Malware Analysis](https://nostarch.com/malware) | Malware | The standard reference for malware analysts |
| [The Art of Memory Forensics](https://www.amazon.com/Art-Memory-Forensics-Detecting-Malware/dp/1118825098) | DFIR | Definitive guide to Volatility and memory analysis |
| [Bug Bounty Bootcamp (Vickie Li)](https://nostarch.com/bug-bounty-bootcamp) | Web/Bug Bounty | Best practical introduction to bug hunting |
| [The Hacker Playbook 3 (Peter Kim)](https://www.amazon.com/Hacker-Playbook-Practical-Penetration-Testing/dp/1494932636) | Red Team | Practical offensive techniques with lab exercises |
| [Penetration Testing (Georgia Weidman)](https://nostarch.com/pentesting) | Pentesting | Great lab-based beginner-to-intermediate guide |
| [Computer Systems: A Programmer's Perspective (CS:APP)](http://csapp.cs.cmu.edu/) | Systems | How computers actually work at the hardware level |
| [Operating Systems: Three Easy Pieces](https://ostep.org/) | OS Internals | Free, excellent OS fundamentals |
| [Cryptography Engineering (Ferguson, Schneier, Kohno)](https://www.amazon.com/Cryptography-Engineering-Principles-Practical-Applications/dp/0470474246)
