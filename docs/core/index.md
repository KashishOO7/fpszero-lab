# Identity ∅

## What is fpszero?

**fpszero** is the lab of **Kashish** — a security researcher, systems thinker, and builder operating from first principles.

The name encodes the philosophy:

- **fps** → *First Principles Synthesis* — break any system to its base axioms before rebuilding understanding from the ground up.
- **zero** → The origin. Zero is not empty. Zero is the reference point from which all measurement begins.

This is not a portfolio. It is not a blog. It is a **knowledge engine** — a living document of how a security-oriented mind maps, breaks, and rebuilds complex systems.

---

## Operating Philosophy

> *"If I can't explain it simply, I don't understand it yet."*

Most technical content falls into two failure modes:

1. **Too shallow** — certification dumps, surface-level tutorials, tool walkthroughs with no underlying theory.
2. **Too fragmented** — brilliant research buried in PDFs, Discord servers, or behind paywalls that a beginner can never find.

fpszero-lab is the middle layer: **deeply understood fundamentals, synthesized into navigable knowledge.**

The scope is not limited to one domain. Security touches systems, networks, hardware, physics, mathematics, mobile, cloud, cryptography, and everything in between. If it has a foundation that can be decomposed to first principles — it belongs here.

### The Three Laws of this Lab

1. **Verify before you trust.** Everything curated here is filtered — but the internet changes. Apply your own judgment.
2. **Understand before you execute.** Copy-paste security is not security. Know what the command does.
3. **Build to break. Break to learn.** Experimentation is the only real teacher.
