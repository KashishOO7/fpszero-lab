# Contributing to fpszero-lab

## Reporting Issues

- Broken links → open a GitHub issue with the page URL and broken link
- Factual errors → open an issue with the correction and a source
- Resource suggestions → open an issue with the resource, domain, and why it belongs

## Pull Requests

PRs are welcome for:

- Broken link fixes
- Typo corrections
- Resource additions (with justification)

PRs should not:

- Add content without sources or citations
- Add tools or resources without testing/verification
- Change the structure or philosophy of the lab

## Style

- No filler. Every sentence should carry information.
- External links must be verified at time of submission.
- Code examples must be tested.
- No fabricated output or results.
